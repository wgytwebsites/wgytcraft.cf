## How to use
### Require the module
```javascript
var wgytcraft = require('@wgytcraft/hosting').main
```
### Use it: 
```javascript
wgytcraft(modulesJson, dirname)
// modulesJson should be the modules.json file for wgytcraft
// dirname should be __dirname

```
